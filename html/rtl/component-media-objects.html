Media Alignment

The images or other media can be aligned top, middle, or bottom. The default is top aligned.

For left align, use class .media-left, For Right align, use class .media-right.

Now, for vertical align, Top is default alignment. For Middle alignment, use .media-middle and For Bottom alignment, use .media-bottom with left or right aligned class.
